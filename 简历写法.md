# 简历写法 — 路径规划项目

## 1. 项目名称（任选其一）
- 朴实版：**移动机器人路径规划可视化仿真器**
- 略高级版：**基于 A\* 的移动机器人路径规划与避障可视化仿真系统**
- 英文版：**Mobile Robot Path Planning & Obstacle-Avoidance Visual Simulator**

## 2. 一句话描述（任选其一）
- 用 Python 实现 A\*/Dijkstra/贪婪三种路径规划算法，在交互式栅格地图上可视化机器人搜索与最短路径行走过程。
- 独立开发的移动机器人全局路径规划演示系统，自实现 A\* 算法并配 pygame 实时动画与算法效率对比。

## 3. 项目要点（写进简历正文）

**精简版（空间有限时，2 条）：**
- 基于 Python 独立实现 A\*、Dijkstra、贪婪最佳优先三种路径规划算法，在交互式栅格地图（40×25）上可视化搜索与避障过程，支持算法实时切换与「探索节点数」效率对比。
- 采用算法与界面解耦的工程结构，为核心算法编写 4 个单元测试用例保证正确性。

**详细版（4 条）：**
- 独立设计并实现移动机器人全局路径规划仿真器，自实现 A\* 算法（f=g+h，曼哈顿启发式，heapq 优先队列），在含障碍栅格地图上求解最短无碰撞路径。
- 实现 A\*、Dijkstra、贪婪最佳优先三种算法的统一搜索框架，通过对比「探索节点数」直观展示 A\* 的效率优势。
- 使用 pygame 开发交互式可视化界面，支持鼠标实时编辑障碍、设置起终点，并以动画演示搜索扩展、最短路径与机器人行走全过程。
- 将算法核心与界面解耦为独立模块，编写单元测试覆盖直线、绕障、无解等场景，保证算法正确性。
