如何录制演示 GIF（招聘会要用，强烈建议做）
==========================================

为什么要做：
  招聘会现场你不一定有时间打开电脑跑程序，手机里存一个 10-20 秒的演示 GIF /
  短视频，递给 HR 或面试官看，效果最直接。GitHub 的 README 里也会用到它。

录什么内容（建议剧本，20 秒左右）：
  1. 打开程序（python app.py），地图是空的。
  2. 鼠标左键拖几道障碍 / 画一个简单迷宫。
  3. 按 S 点一下设起点，按 G 点一下设终点。
  4. 按空格 —— 让它播放“搜索动画 + 找到最短路 + 机器人走过去”。
  5. 按 2 切到 Dijkstra 再按空格，对比一下 expanded(探索格子数)，
     说明 A* 探索得更少、更聪明。

Windows 录 GIF 的两种简单方法：
  方法一（推荐，免费）：ScreenToGif
    - 官网 https://www.screentogif.com/ 下载，打开选“录像机”，
      框住程序窗口 → 录制 → 停止 → 直接导出 GIF。
  方法二（系统自带，录视频）：Win + G 打开 Xbox Game Bar → 录制窗口 → 得到 mp4，
    再用 ScreenToGif 或在线工具转成 GIF。

做好后：
  把文件命名为 demo.gif，放到本目录（docs/）下，
  README.md 里的 ![demo](docs/demo.gif) 就能自动显示。
